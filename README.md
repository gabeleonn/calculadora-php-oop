# Calculadora em PHP OOP
> Simples calculadora orientada a objetos usando regex.

## Uso

inicie pelo terminal/prompt com:
```
php main.php
```
Digite sua formula como uma string. Com ou sem espaços funciona. 
> Ex: 1+2 ou 1 * 10
